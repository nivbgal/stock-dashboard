HOW TO RUN
================================================================================

1. Navigate to project:
   cd stock-dashboard

2. Install dependencies:
   npm install

3. Set up environment (optional for demo):
   cp .env.example .env.local
   # Edit with your API keys

4. Start development server:
   npm run dev

5. Open browser:
   http://localhost:5173
